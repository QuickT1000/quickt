## Environments
You need to rename .env.dist to .env
~~~bash
mv ./.env.dist .env
~~~

## Install Project

Go to root folder of the cloned project and install the deplendencies:

~~~bash
./build/build.sh
~~~

## Docker
To start up the application run:
~~~bash
docker-compose up
~~~
